# Этап 9. Юридическая и техническая спецификация формы заявки

## Назначение формы
Получить минимально необходимые данные для связи с пользователем и обсуждения его запроса.

## Обязательные поля
- name — Имя
- contact — Телефон или email
- message — Описание задачи или объекта
- consent_pd — обязательный чекбокс согласия на обработку персональных данных

## Необязательные поля
- city — Город
- asset_type — Тип недвижимости
- budget — Бюджет

## Технические требования
- серверная валидация
- honeypot anti-spam
- ограничение частоты отправки
- логирование факта согласия
- редирект на /thanks
- HTTPS обязательно

## Текст у чекбокса
Нажимая «Отправить», я даю согласие на обработку моих персональных данных для связи со мной и обсуждения моего запроса. Политика: /privacy

## Что недопустимо
- предустановленный чекбокс
- объединять согласие на ПДн с другими согласиями
- собирать лишние данные без цели
- отправлять форму без ссылки на политику
