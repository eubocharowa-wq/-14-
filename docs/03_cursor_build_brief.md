Новый проект с нуля.
Не редизайн старого лендинга.

## Что важно на старте
- чистая архитектура
- mobile-first
- отсутствие перегруженной верхней строки
- понятная навигация

## На этом этапе делаем
- структуру проекта
- базовую навигацию
- каркас страниц
- общий layout
- адаптивную шапку
- чистый mobile hero
